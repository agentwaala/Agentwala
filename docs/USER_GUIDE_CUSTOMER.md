# Customer User Guide

## Getting Started

### Creating an Account

1. Click **"Get Started"** or **"Log in"** in the navbar
2. Sign in with your Google account
3. On first login, select **"Customer"** as your role
4. You'll be redirected to your Customer Dashboard

### Finding Agents

#### Method 1: Browse All Agents
1. Click **"Find Agents"** in the navigation
2. Use filters to narrow down:
   - **Search**: Type agent name or expertise
   - **Domain**: Select category (Real Estate, Clothes, etc.)
   - **Location**: Filter by region

#### Method 2: Near You Discovery
1. Go to the homepage
2. Scroll to **"Agents Near You"** section
3. Click **"Auto-detect My Location"** (requires GPS permission)
4. Or manually select:
   - State
   - District
   - Locality
5. Sort by: Nearest, Highest Rated, Verified First, Available Now

### Agent Profiles

Each agent card shows:
- **Name** and profile photo
- **Blue tick** = Verified agent
- **Gold badge** = Premium agent
- **Green dot** = Currently available
- **Grey dot** = Currently unavailable
- **Category** (e.g., Real Estate, Textiles)
- **Location** with distance from you
- **Star rating** (visual only, no numbers)

### Contacting Agents

1. Click on an agent card to view full profile
2. If agent is **Available** (green dot):
   - Click **"Call Now"** for immediate contact
   - Click **"Schedule Call"** for later
3. If agent is **Unavailable** (grey dot):
   - Contact button is disabled
   - You can still save the agent

### Rating Agents

After a call interaction:
1. Go to **Dashboard → Call History**
2. Find the call you want to rate
3. Click **"Rate"** button
4. Select 1-5 stars
5. Rating is submitted immediately

**Rules:**
- Only star ratings allowed (no text)
- One rating per agent interaction
- Ratings cannot be edited once submitted

### Saving Agents

1. View an agent's profile
2. Click the **heart icon** to save
3. Access saved agents from **Dashboard → Saved Agents**

### Customer Dashboard

Access by clicking your profile avatar → **Dashboard**

#### Sections:

**Call History**
- View all past calls
- See agent name, category, date, duration
- Rate unrated calls

**Saved Agents**
- Quick access to favorite agents
- See online/offline status
- Click to view full profile

**Profile Card**
- Your name and email
- Total calls made
- Number of saved agents

**Location Preferences**
- Set preferred state
- Quick access to nearby agents

## FAQ

**Q: Why can't I contact an agent?**
A: The agent is currently unavailable (offline). Try saving them and checking back later.

**Q: Can I change my rating?**
A: No, ratings are final once submitted.

**Q: Why don't I see any agents near me?**
A: Either no agents are registered in your area, or GPS permission was denied. Try selecting location manually.

**Q: Is there a subscription fee?**
A: No, the platform is currently free for all customers.
